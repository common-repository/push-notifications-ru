Push Нотификации для Wordpress. Присылает сообщения на рабочий стол ПК или мобильный

== Description ==

Плагин «Push Нотификации RU» – интеграция решения для рассылки push-нотификаций Pushem.ru:

* БЕСПЛАТНЫЙ ТАРИФ,
* Безлимитная отправка сообщений,
* Геотаргетинг (Страна, Город),
* Сегментация,
* UTM-метки,
* Простая и быстрая установка.

**Преимущества push-нотфикаций**

Push-нотификации это мощнейший маркетинговый инструмент, который дает возможность вам отправлять пользователям короткие сообщения прямо на рабочий стол, по клику на которые они вернутся на ваш сайт

* Возвращаются до 70% пользователей на сайт,
* Увеличивают продажи до 10 раз,
* Уменьшают стоимость клиента на 300%,
* Действительно дешевый ремаркетинг,
* В 100 раз эффективнее e-mail рассылок,
* В разы быстрее SEO

**Настройка плагина**

Чтобы включить push-нотификации на вашем сайте, нужно установить плагин на ваш WP-сайт, создать [аккаунт на сайте Pushem.ru](https://pushem.ru/), ввести название вашего сайта и перейти в разделе "CMS плагины".
На этой странице вы найдете нужные данные для настройки.
Вставьте эти данные в поля ID и ключ API в настройках плагина в административной части вашего WP-сайта. Затем нажмите сохранить.
Вставьте эти данные в поля ID и ключ API в настройках плагина в административной части вашего WP-сайта.
Страница настроек плагина Push-нотификаций находится в WP-admin -> Настройки -> Push Нотификации
При возникновении ошибок или сложностей свяжитесь с технической поддержкой по e-mail: support@pushem.ru и вам обязательно помогут.
